## 要点

1. 开源的 AI 股票深度分析插件，全免费数据源，零 API key
2. 覆盖 A 股 / 港股 / 美股，尤其强调对中国市场数据的深度支持
3. 22 维度数据 × 180+ 条量化规则 × 22 种机构级估值方法
4. 独创 "66 位 AI 大佬评审团" × 9 大流派（巴菲特、索罗斯、赵老哥、a16z、马斯克等）
5. 输出 Bloomberg 风格的 HTML 报告 + 朋友圈竖图 + 微信群战报
6. 支持 Claude Code、Cursor、Gemini CLI、OpenClaw 等多个 AI Agent 平台
7. 核心命令：analyze-stock（完整分析）/ quick-scan（速判）/ scan-trap（杀猪盘排查）/ dcf（估值专项）
8. 三档分析深度：lite（30-60s）/ medium（2-4min）/ deep（5-8min+ 含 AI 评委角色扮演）
9. Serenity AI 卡位猎手组：8 罚分因子 + 3 级证据阶梯 + 供应链 8 层分层
10. v3.7 新增 Hermes 安装支持 + v3.8 严谨化升级 + v3.9 HTML 报告导航优化
