# RAG 作为 Agent 组件

来源：微信公众号"算法狗"（腾讯面试官对话）
原文：https://mp.weixin.qq.com/s/IzheS512kExzwXzRy-fBiw
日期：2026-07-05
摘要：RAG 没有消失，从独立系统降级为 Agent 架构下的一个组件。

核心观点：
- RAG 不是死了，是从"独立系统"降级成了"一个组件"
- 在更大的 Agent 架构下，RAG 是 Agent 工具箱中的一个能力，与代码解释器、API 调用等并列
- Agent 作为"大脑"，决定何时以及如何使用检索能力
- Agentic RAG 可以动态进行多次检索，根据需要调用不同的 RAG 工具
- RAG 提供了不可替代的价值：处理海量非结构化数据、实时知识更新、高合规性溯源
- 这种演进使 AI 系统能处理更复杂的查询，进行多步推理
