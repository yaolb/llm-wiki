# MUX0

## 概述

MUX0 是专为 Claude Code 设计的开源 AI 编程终端，基于 Ghostty 终端框架，为 AI Agent 提供优化的命令行交互环境。

## 核心信息
- **全称**：MUX0
- **类型**：AI 编程终端
- **技术栈**：基于 Ghostty
- **定位**：Claude Code / OpenCode / Codex 专用终端

## 核心特性

- 为 AI Agent 工具调用优化（文件读写、代码搜索、Bash 执行）
- 支持多 Agent 会话管理
- 原生集成 Claude Code 的工具链
