# CC-Connect

## 概述

CC-Connect 是一个桥接工具，让用户可以在飞书、微信等即时通讯应用中直接调用 Claude Code 等 AI Agent 进行编程，将 AI 编程能力融入日常沟通协作场景。

## 核心信息
- **全称**：CC-Connect
- **类型**：AI Agent 通讯桥接工具
- **支持平台**：飞书、微信等
- **对接后端**：Claude Code / OpenClaw 等

## 适用场景

- 在飞书群聊中 @Agent 执行代码审查
- 在微信中发送代码片段让 Agent 分析
- 非技术用户通过聊天界面使用 AI 编程能力
