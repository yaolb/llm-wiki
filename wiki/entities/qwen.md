# Qwen

## 概述

Qwen（通义千问）是阿里巴巴集团旗下的阿里云（Alibaba Cloud）和达摩院开发的大语言模型系列。Qwen 系列覆盖从 0.5B 到 110B 的参数规模，支持中文多模态，在中文场景下表现优异。

## 核心信息
- **开发者**：阿里云 / 达摩院
- **代表模型**：Qwen（7B、14B、72B）、Qwen2、Qwen2.5
- **GitHub**: [QwenLM/Qwen](https://github.com/QwenLM/Qwen)
- **官网**: [qwen.aliyun.com](https://qwen.aliyun.com)
- **开放协议**：Apache 2.0 / Qwen License

## 特点
- 中文能力突出，适合中文企业应用
- 多模态支持（文本 + 图像 + 视频）
- 系列完整（0.5B ~ 110B、Dense 和 MoE 架构均有）
- 支持工具调用（Function Calling）和 Agent 场景
